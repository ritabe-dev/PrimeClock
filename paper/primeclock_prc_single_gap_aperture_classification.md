# A Single-Gap Aperture Classification in a Prime-Indexed Circle-Arc Covering Model

## Abstract

We study a finite prime-indexed circle-arc covering model generated by residue
data.  Fix an old prime prefix up to `p_k`, an old residue class
`r in Z/M_kZ`, and a later odd prime modulus `q>p_k`.  The main result
classifies exactly when adding one direct `q`-arc can create a nonempty
terminal birth: this happens precisely when the old residual set has a single
gap and the `q`-grid is aligned with the corresponding aperture window.

The result is internal to the PRC circular-arc model.  It is not a theorem about
prime gaps, not an asymptotic law, not a general predictor, and not a theorem
about classical covering systems of congruences.

## 1. Definitions

Let `T=R/Z`.  Fix `k>=3`, let the old prime prefix be

```text
p_1=2, p_2=3, ..., p_k,
M_k = product_{i<=k} p_i,
```

and fix an old residue `r in Z/M_kZ`.  For each old prime `p_i`, write
`b_i = r mod p_i` and define the old `p_i`-arc by

```text
I_{p_i}(r) = [ (b_i - 1/2)/p_i, (b_i + 1/2)/p_i ] subset T.
```

Let

```text
U_k(r) = union_{i<=k} I_{p_i}(r),
R_k(r) = T \ U_k(r).
```

The connected components of `R_k(r)` are the old residual gaps.  Let `q` be any
odd prime with `q>p_k`.  For a remainder `a in Z/qZ`, define the new `q`-arc

```text
I_q(a) = [ (a - 1/2)/q, (a + 1/2)/q ] subset T.
```

The theorem concerns the direct one-prime q-lift over the old prefix.  When
`q != p_{k+1}`, it does not claim that intermediate sequential PRC transitions
are skipped or unchanged.

## 2. Main theorem

### Theorem

Fix `k>=3`, an old residue `r in Z/M_kZ`, and an odd prime `q>p_k`.  In the PRC
circular-arc model:

1. If `R_k(r)` is empty, the old row is already terminally closed, so no
   nonempty `q`-birth is counted.
2. If `R_k(r)` has at least two connected components, no `q`-remainder produces
   a closed or strict nonempty `q`-birth lift.
3. If `R_k(r)` has exactly one connected component `G=(L,R)`, then a
   `q`-remainder `a` produces a nonempty `q`-birth lift exactly when there is a
   lifted integer representative `tilde a in a + qZ` satisfying

   ```text
   qR - 1/2 < tilde a < qL + 1/2.
   ```

The open interval

```text
A_q(G) = (qR - 1/2, qL + 1/2)
```

is the aperture window.  It contains at most one integer, so a single old
residual gap has at most one `q`-remainder candidate.

## 3. Proof

The proof is geometric proof inside the stated circle-arc model.

### 3.1 Abstract short-arc obstruction

Let `U` be a finite union of closed arcs on `T`, and let `R=T\U`.  Suppose every
nonempty connected component of `U` has length at least `delta`.  If a closed
arc `J` has length `lambda<delta`, then `J` cannot contain closures of two or
more connected components of `R`.

Choose a cut outside `J` and work in the resulting local linear representative.
If two residual components inside `J` are adjacent in that linear order, the
closed interval between them lies in one connected component of `U`.  That
covered component has length at least `delta`, but it would have to fit inside
`J`, whose length is `lambda<delta`.  This is impossible.

### 3.2 PRC component lower bound

Every nonempty connected component of `U_k(r)` contains at least one old
`p_i`-arc.  Since `p_i<=p_k`, that arc has length `1/p_i >= 1/p_k`.  Therefore
every nonempty covered component of `U_k(r)` has length at least `1/p_k`.

### 3.3 Multi-gap obstruction

The new `q`-arc has length `1/q`.  Since `q>p_k`,

```text
1/q < 1/p_k.
```

Apply the abstract short-arc obstruction with `delta=1/p_k` and `lambda=1/q`.
One `q`-arc cannot contain closures of two or more old residual gaps.  Hence a
multi-gap old residual set cannot produce a nonempty terminal `q`-birth.

### 3.4 Closed containment gives no extra births

The proof first classifies strict terminal containment.  In this model, closed
terminal containment gives no additional nonempty birth cases.  If a closed
containment were not strict, a `q`-arc endpoint would touch the closure of the
old residual set.  Such a point is an old endpoint.  Old/new endpoint
coincidence for an old prime `p<=p_k` and a later odd prime `q>p_k` can only
occur at the central point `1/2`.  The corresponding central remainders are
non-birth remainders, so closed-only endpoint-touch birth cases do not occur.

### 3.5 Single-gap aperture criterion

Assume `R_k(r)` has exactly one residual gap `G=(L,R)`.  Work in the universal
cover with lifted real coordinates `L<R`.  If the gap crosses the `0`-cut in
`T`, the chosen lift may have `R>1`.

A lifted `q`-arc contains the closure of `G` strictly exactly when

```text
(tilde a - 1/2)/q < L
and
R < (tilde a + 1/2)/q.
```

Equivalently,

```text
qR - 1/2 < tilde a < qL + 1/2.
```

This is the aperture inequality.  The aperture window has width

```text
1 - q length(G).
```

If this width is non-positive, the window is empty.  If it is positive, then
`length(G)>0`, so the width is strictly less than `1`.  Therefore the aperture
window contains at most one integer.

## 4. Computational sanity checks

The theorem proof is not the code.  The repository separates the mathematical
argument from implementation and distribution checks:

| Item | Role |
| --- | --- |
| Theorem note | geometric proof of the model theorem |
| Exact rational sanity check | finite sanity check of definitions and examples |
| Recorded birth rows consistency audit | confirms committed CSV birth rows agree with the next-prime corollary |
| Release integrity checks | verifies manifest, DOI, bundle, and hash consistency |

In particular, the recorded birth rows consistency audit is not a full
finite-universe completeness audit and is not the proof of the theorem.

## 5. Related work and non-claims

This paper uses residues, prime moduli, circle arcs, and covering language, but
it is not a result about classical covering systems of congruences.  It studies
a project-defined finite prime-prefix circle-arc model.

The release does not claim:

- no B8 theorem;
- no full transition-graph theorem;
- no general predictor;
- no asymptotic law;
- no prime-gap theorem outside PRC model;
- not a full finite-universe completeness audit;
- no mechanically verified proof.

The intended contribution is a structural classification inside the PRC
circular-arc model: nonempty q-births are single-gap aperture-alignment events.
