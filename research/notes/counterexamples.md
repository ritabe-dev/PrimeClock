# Counterexamples and Pitfalls

This file records mistakes the project must avoid.

## Endpoint Atom

If `p | N`, then

```text
Phi_N(p) = 0
```

So finite `N` can have mass at `x=0`. This atom disappears in the limit, but it
can affect small experiments.

## Not Uniform

The limiting density is not uniform. It is larger near `x=0`.

## Limit Versus Finite N

The limiting density `rho` is not always the best finite-N comparison. The
finite-N PNT model may explain small and medium `N` better.

## Visual Evidence Is Not Proof

The clock image and histogram figures can suggest structure, but claims need
either proof or explicit experiment status.

## Naming

Do not present the research as `PrimeClock`. `PrimeClock / Prime Clock` has
existing collisions and should be treated as the old visual prototype name.
