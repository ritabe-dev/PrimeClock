# Prime Reciprocal Projection v0.1

Prime Reciprocal Projection (PRP) は、旧 PrimeClock の視覚化から切り出した
研究用の記法である。PrimeClock は、素数秒に新しい周期針が生まれ、
それぞれの素数 `p` の針が `p` 秒で一周する時計だった。その中心にある
数式は、時刻 `N` における素数 `p` の位相

```text
{N / p}
```

である。

PRP は、この位相だけを取り出し、素数を円周上の点配置として読むための
実験数学的な枠組みである。現段階の目的は、新しい素数分布定理を主張する
ことではない。目的は、定義を固定し、数値実験を再現可能にし、どの主張が
既知結果の範囲に入りそうかを整理することである。

## 1. 定義

整数 `N >= 2` を固定する。素数 `p <= N` に対して、

```text
Phi_N(p) = {N / p}
```

と定義する。ここで `{x}` は小数部分であり、値は `[0,1)` に入る。

このとき、PRP の経験測度を

```text
mu_N = (1 / pi(N)) * sum_{p<=N, p prime} delta_{Phi_N(p)}
```

と置く。これは `[0,1)` 上の確率測度である。直感的には、`N` を固定したとき、
すべての素数 `p <= N` を `Phi_N(p)` の位置に置いた点分布である。

PrimeClock の言葉で言えば、これは各素数針の中心位相を集めたものに対応する。
ただし、PRP では時計の針の幅や発光表現はいったん捨て、中心位相だけを見る。

## 2. 極限密度

PRP の最初の観察対象は、`mu_N` が一様分布ではなく、次の密度へ近づくように
見えることである。

```text
rho(x) = sum_{k>=1} 1 / (k + x)^2,  0 <= x < 1
```

この密度は正規化されている。実際、

```text
int_0^1 rho(x) dx
= sum_{k>=1} int_0^1 1/(k+x)^2 dx
= sum_{k>=1} (1/k - 1/(k+1))
= 1
```

である。

重要なのは、この密度が一様ではないことだ。`x=0` 付近で大きく、`x=1` に
向かって下がる。つまり、PRP の位相分布には自然な偏りがある。

ただし、この極限密度そのものを新しい発見として主張するべきではない。
これは素数定理と変数変換から出る自然な対象であり、`{x/n}` や関連列の
既知研究と接続している。

特に、Saffari--Vaughan の
“On the fractional parts of `x/n` and related sequences” は、この方向の
直接的な先行研究である。Part I は、数え上げ関数が正則変動する集合 `A`
に対する `{x/a}` の極限分布を扱っており、素数集合は `pi(x) ~ x/log x`
によりその射程に入る。この対応では、極限分布関数は

```text
F(a) = sum_{k>=1} (1/k - 1/(k+a))
```

となり、その導関数が

```text
F'(a) = sum_{k>=1} 1/(k+a)^2 = rho(a)
```

である。Part II は `h(n)=1/n` を詳しく扱い、さらに素数版も明示的に
扱っている。したがって、PRP v0 では

```text
mu_N => rho(x) dx
```

を新規定理候補ではなく、既知結果への接続対象として扱う。

これは何を意味するか。PRP の基本図形は、素数の未知のランダム性が直接
作っているというより、`p -> {N/p}` という逆数写像が作る双曲線的な
折り畳みでほぼ決まっている。素数定理によって、各分枝には一階近似として
十分な密度が入り、その分枝を足すと `rho` が出る。したがって、数学的に
面白い残りは「極限密度そのもの」ではなく、有限 `N` でそこからどれだけ、
どの形でずれるかである。

## 3. 分枝分解

この密度がどこから出てくるかを見るには、

```text
k = floor(N / p)
```

で分枝に分けるのがよい。

分枝 `k` は

```text
B_{N,k} = { p prime : N/(k+1) < p <= N/k }
```

である。この分枝上では

```text
N / p = k + x
x = {N / p}
p = N / (k + x)
```

となる。したがって、`x` の小さな区間に入る素数を数える問題は、
`p` が `N/(k+x)` 型の区間に入るかどうかを数える問題へ変わる。

極限的には、分枝 `k` の質量は

```text
w_k = int_0^1 1/(k+x)^2 dx
    = 1/(k(k+1))
```

になる。この分枝質量を足し合わせると、全体の密度 `rho(x)` が現れる。

この分枝分解は、PRP を理解する上でかなり重要である。PRP は単に素数を
円周上へ散らしているのではない。`floor(N/p)=k` という双曲線的な層を
重ねている。

## 4. 数値実験 v0

v0 では、以下の値で実験した。

```text
N = 10^3, 10^4, 10^5, 10^6
bins = 100
Fourier modes = 1..20
```

生成済みの主図は次の3つである。

```text
figures/v0/distribution_N100000_bins100.png
figures/v0/branches_N100000_k20.png
figures/v0/fourier_N100000_m20.png
```

### 4.1 分布図

`N=100000` の分布図では、経験分布が極限密度 `rho` の形にかなり沿っている。
棒グラフには有限 `N` の揺れが残るが、全体として `x=0` 付近が大きく、
`x=1` に向かって下がる形が見える。

この図から言ってよいことは、`{N/p}` の経験分布が一様ではなく、`rho` の
形をよく反映しているということである。

一方で、この図だけから「新しい素数分布法則を発見した」とは言わない。
これは視覚的・数値的な確認であり、既知理論との関係を調べる必要がある。

### 4.2 分枝図

分枝図では、`floor(N/p)=k` の経験質量が、理論的な極限値

```text
1 / (k(k+1))
```

に近いことが見える。

この図は、分布図以上に説明力がある。なぜなら、`rho` の形が単に経験的に
見えているだけではなく、分枝 `k` の重ね合わせとして説明できるからである。

ただし、分枝質量は有限 `N` の素数区間数に敏感であり、収束は分布全体より
遅く見える。ここは有限 `N` 補正を考える余地がある。

### 4.3 Fourier 図

Fourier 診断では、

```text
hat_mu_N(m)
= (1 / pi(N)) * sum_{p<=N} exp(-2*pi*i*m*{N/p})
```

と、極限密度 `rho` の Fourier 係数との差を見ている。

`N=100000` では、低周波の残差は小さいが、モードごとに揺れがある。
この段階では、特定モードの山に数論的意味があるとは言わない方がよい。
Fourier は、現時点では「分布の低周波成分も極限対象に近づくか」を見る
診断道具として扱う。

この表から、少なくとも v0 の範囲では次のことが言える。

第一に、histogram L1 は `N` が増えるほど大きく下がっている。これは
経験分布が `rho` の形へ近づくという視覚的観察を数値で支えている。

第二に、KS distance も単調に下がっている。したがって、分布図の一致は
100 bins の取り方だけによる偶然ではなさそうである。

第三に、Fourier 平均残差も下がっている。これは、低周波成分という別の
見方でも、経験分布が極限密度へ近づくことを示唆している。

第四に、branch L1 の下がり方は比較的遅い。これは、各分枝が有限 `N` の
素数区間数に敏感であるためだと考えられる。分枝分解は極限構造の説明として
強いが、有限 `N` では PNT 補正モデルとの比較が必要になる。

## 6. 何を主張しないか

v0 では、次のことを主張しない。

- PRP が新しい素数分布法則を発見したとは言わない。
- 素数が円周上に一様ランダムに並ぶとは言わない。
- 極限密度 `rho` そのものが新規だとは言わない。
- 弱収束 `mu_N => rho(x) dx` そのものが新規だとは言わない。
- 図や時計の見た目を証明の代わりにはしない。
- Fourier の特定モードの山に、現時点で数論的意味があるとは言わない。

この研究の現時点での価値は、PrimeClock から出てきた位相

```text
{N / p}
```

を、再現可能な実験対象として整理したことにある。極限密度、分枝分解、
Fourier 診断、有限 `N` 補正という形で、次の問いを立てやすくなった。

## 7. 次の問い

次に調べるべき問いは次の通りである。

1. `mu_N => rho(x) dx` は、どの既知定理の系として引用すべきか。
2. 有限 `N` では、`rho` と PNT 補正モデル `rho_N^PNT` のどちらがどの程度
   よく合うか。
3. histogram L1、KS distance、Fourier residual は、どの程度の速度で減るか。
4. 分枝ごとの誤差は、素数の短区間分布として説明できるか。
5. PrimeClock の針幅や被覆を戻したとき、中心位相のPRPからどのような
   追加量が自然に出るか。

## 8. 現時点の位置づけ

PRP v0.1 は、完成した数学論文ではない。これは、外部レビューに出す前の
研究artifactである。

外部に聞くべき最初の質問は、次のように絞るのがよい。

```text
The empirical measure

mu_N = (1/pi(N)) sum_{p<=N} delta_{{N/p}}

appears to converge to rho(x) dx, where

rho(x) = sum_{k>=1} 1/(k+x)^2.

What is the correct existing reference for this weak convergence over primes?
Are the finite-N correction and Fourier diagnostics reasonable objects to study?
```

既知結果チェックの現時点での答えは、Saffari--Vaughan I/II を第一候補として
引用する、である。次の段階では、PRP の記法と Saffari--Vaughan の記法を
1対1で対応させ、どの定理・系を引用すれば十分かを確定する。
