# Prime Reciprocal Projection: 2ページ要約 v0.1

## 概要

Prime Reciprocal Projection (PRP) は、旧 PrimeClock の視覚化から切り出した
実験数学用の記法である。PrimeClock では、時刻 `N` における素数 `p` の針の
中心位相が

```text
{N / p}
```

として現れる。PRP はこの位相だけを取り出し、素数 `p <= N` を `[0,1)` 上の
点配置として調べる。

現段階では、新しい素数分布定理を主張しない。目的は、定義、極限密度、
分枝分解、数値実験を整理し、既知結果との正しい接続を見つけることである。
既知結果チェックでは、Saffari--Vaughan の `{x/n}` と関連列の研究が
中心的な参照先になる見込みである。

## 定義

整数 `N >= 2` を固定し、素数 `p <= N` に対して

```text
Phi_N(p) = {N / p}
```

と定義する。ここで `{x}` は小数部分であり、値は `[0,1)` に入る。

PRP の経験測度を

```text
mu_N = (1 / pi(N)) * sum_{p<=N, p prime} delta_{Phi_N(p)}
```

と置く。これは `[0,1)` 上の確率測度である。

期待される極限密度は

```text
rho(x) = sum_{k>=1} 1 / (k + x)^2,  0 <= x < 1
```

である。この密度は

```text
int_0^1 rho(x) dx
= sum_{k>=1} (1/k - 1/(k+1))
= 1
```

により正規化されている。

この弱収束そのものは、PRP の新規主張にはしない。Saffari--Vaughan I は
正則変動する数え上げ関数を持つ集合 `A` 上の `{x/a}` を扱い、素数集合は
`pi(x) ~ x/log x` によってこの枠組みに入る。Saffari--Vaughan II は
`h(n)=1/n` の標準ケースと素数版を明示的に扱っている。

意味としては、`rho` は素数の神秘的な新分布ではなく、逆数写像
`p -> {N/p}` の分枝構造から生まれる一階の形である。研究対象として
残るのは、有限 `N` でこの既知極限からどうずれるか、そしてそのずれを
branch/Fourier/PNT補正でどこまで説明できるかである。

## 分枝分解

PRP の構造は

```text
k = floor(N / p)
```

で分枝に分けると見やすい。分枝 `k` は

```text
B_{N,k} = { p prime : N/(k+1) < p <= N/k }
```

である。この分枝上では

```text
N / p = k + x
x = {N / p}
p = N / (k + x)
```

となる。

極限的な分枝質量は

```text
w_k = int_0^1 1/(k+x)^2 dx = 1/(k(k+1))
```

であり、これらの分枝を足し合わせることで `rho(x)` が現れる。

## v0 数値実験

v0 では

```text
N = 10^3, 10^4, 10^5, 10^6
bins = 100
Fourier modes = 1..20
```

で実験した。

| N | pi(N) | histogram L1 | KS distance | branch L1 k<=20 | Fourier mean m<=20 |
|---:|---:|---:|---:|---:|---:|
| 1,000 | 168 | 0.5417 | 0.0384 | 0.1438 | 0.0445 |
| 10,000 | 1,229 | 0.1915 | 0.0241 | 0.0759 | 0.0180 |
| 100,000 | 9,592 | 0.0548 | 0.0131 | 0.0583 | 0.0062 |
| 1,000,000 | 78,498 | 0.0219 | 0.0087 | 0.0488 | 0.0027 |

この表から、少なくとも v0 の範囲では、histogram L1、KS distance、
Fourier 平均残差が `N` とともに下がっている。これは `mu_N` が `rho(x) dx`
へ近づくという中心的な見方を数値的に支えている。

一方で、branch L1 の下がり方は遅い。これは分枝ごとの素数区間数が有限 `N`
ではまだ大きく揺れるためだと考えられる。したがって、有限 `N` では
極限密度 `rho` だけでなく、PNT に基づく補正モデルも比較対象にすべきである。

## 主張しないこと

v0 では、次のことを主張しない。

- 新しい素数分布法則を発見したとは言わない。
- 極限密度 `rho` が新規だとは言わない。
- 弱収束 `mu_N => rho(x) dx` が新規だとは言わない。
- 図や時計の見た目を証明の代わりにはしない。
- Fourier の特定モードの山に数論的意味があるとはまだ言わない。

## 外部レビューで聞きたいこと

最初に確認したい質問は次の2つである。

1. Saffari--Vaughan I/II のどの定理・系を `mu_N => rho(x) dx` の引用元にするのが最も正確か。
2. 有限 `N` 補正、分枝誤差、Fourier 診断は、PRP の実験数学的対象として
   妥当か。

この2点を確認した上で、PRP を「既知理論の可視化・再整理」として進めるのか、
有限 `N` の補正や Fourier 診断を中心にした研究ノートへ進めるのかを判断する。
