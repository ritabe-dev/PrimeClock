# PRC v2.7 General q-Prime Single-Gap Aperture Classification Theorem

**Status:** public theorem release text, v1.0. Zenodo version DOI assigned.
**Scope:** PRC circular-arc model, general over all old residues and all later odd prime moduli `q>p_k`.
**Non-claim:** this note is not a mechanically verified proof, not a predictor, not an asymptotic law, and not a claim about prime gaps outside the stated PRC model.

---

## 1. Definitions

Work on the circle

```text
T = R/Z.
```

Fix `k >= 3`. Let the old prime prefix be

```text
p_1 = 2, p_2 = 3, ..., p_k,
```

and let

```text
M_k = product_{i<=k} p_i.
```

Fix an old residue

```text
r in Z/M_kZ.
```

Let `q` be any odd prime with

```text
q > p_k.
```

The next-prime case `q=p_{k+1}` is a corollary of the general q-prime theorem.
When `q>p_k` is not `p_{k+1}`, this theorem concerns the direct one-prime
q-lift over the old prefix. It does not claim that intermediate sequential PRC
transitions are skipped or unchanged.

For each old prime `p_i`, write

```text
b_i = r mod p_i,       b_i in {0,1,...,p_i-1}.
```

The old `p_i`-arc is the closed circular interval

```text
I_{p_i}(r) = [ (b_i - 1/2)/p_i, (b_i + 1/2)/p_i ]  subset T.
```

Equivalently, `I_{p_i}(r)` is the closed arc centered at `b_i/p_i` with radius
`1/(2p_i)`. Its circular length is `1/p_i`.

Define the old covered set and old residual set by

```text
U_k(r) = union_{i<=k} I_{p_i}(r),
R_k(r) = T \ U_k(r).
```

Then `U_k(r)` is a finite union of closed arcs, and `R_k(r)` is an open subset
of `T`. The connected components of `R_k(r)` are called old residual components.

For a q-remainder

```text
a in Z/qZ,
```

define the new q-arc

```text
I_q(a) = [ (a - 1/2)/q, (a + 1/2)/q ] subset T.
```

When a circular interval crosses a chosen cut, use a local linear representative.
Equivalently, use a lifted integer representative

```text
tilde a in a + qZ
```

and write

```text
I_q(tilde a) = [ (tilde a - 1/2)/q, (tilde a + 1/2)/q ] subset R
```

inside the chosen local representative.

## 2. Terminal containment and birth terminology

This note distinguishes three notions.

### Component strict q-containment

A residual component `G` is strictly q-contained by `a` if

```text
closure(G) subset int(I_q(a)).
```

### Row-level strict terminal containment

The lifted row `(r,a)` is strictly q-terminal if

```text
closure(R_k(r)) subset int(I_q(a)).
```

If additionally `R_k(r)` is nonempty, this note calls `(r,a)` a strict q-birth
lift.

### Row-level closed terminal containment

The lifted row `(r,a)` is closed q-terminal if

```text
closure(R_k(r)) subset I_q(a).
```

If additionally `R_k(r)` is nonempty, this is the closed-containment analogue of
a q-birth lift.

This note proves the strict classification first, then proves that, in the
stated later-prime arc model, closed terminal containment gives no additional
nonempty birth cases beyond the strict classification.

If earlier PRC artifacts use `Close(row)` for this same closed terminal
containment test, then the closed-containment corollary below applies to that
usage. If an artifact uses a different implementation-level definition of
`Close(row)`, that implementation should be audited separately against the
mathematical definition above.

## 3. Endpoint spacing near 0 and 1/2

### Lemma 3.1: old endpoint spacing

For old endpoints generated by primes up to `p_k`,

```text
dist(0, nearest old endpoint) >= 1/(2p_k),
dist(1/2, nearest old endpoint other than 1/2) >= 1/p_k.
```

### Proof

For an odd old prime `p`, every endpoint of an old p-arc has the form

```text
m/(2p),       m in {1,3,...,2p-1}.
```

The closest such endpoints to `0` have distance `1/(2p)`. Since `p <= p_k`, the
minimum over the old odd primes is at least `1/(2p_k)`. For `p=2`, the endpoints
are `1/4` and `3/4`, which are farther from `0` than `1/(2p_k)` because `k>=3`
and hence `p_k>=5`.

For an odd old prime `p`, the endpoint `1/2` occurs when `m=p`. The nearest other
old endpoints occur at `m=p-2` and `m=p+2`, at distance `1/p` from `1/2`. Taking
`p <= p_k` gives distance at least `1/p_k`. For `p=2`, the endpoints `1/4` and
`3/4` are at distance `1/4` from `1/2`, again larger than or equal to the required
bound for `p_k>=5`. ∎

## 4. Forbidden special remainders

### Lemma 4.1: the zero remainder is non-birth

The q-remainder

```text
a = 0
```

does not produce a closed or strict q-birth lift.

### Proof

Use the local representative

```text
I_q(0) = [-1/(2q), 1/(2q)].
```

By Lemma 3.1, the nearest old endpoint to `0` is at distance at least
`1/(2p_k)`. Since `q > p_k`,

```text
1/(2q) < 1/(2p_k).
```

Therefore `I_q(0)` contains no old endpoint. On the local interval from `0` to
the nearest old endpoint on either side, membership in `U_k(r)` and `R_k(r)` is
constant.

If a side inside `I_q(0)` is covered, it contains no residual component. If a
side is uncovered, the residual component on that side extends beyond `I_q(0)`
to the nearest old endpoint and therefore is not contained in `I_q(0)`. Any
residual component outside this local neighborhood is also outside `I_q(0)`. If
there is no residual component anywhere, then there is no birth. Hence `a=0`
cannot produce a closed or strict q-birth lift. ∎

### Lemma 4.2: the two central remainders are non-birth

The q-remainders

```text
a = (q-1)/2,     a = (q+1)/2
```

do not produce closed or strict q-birth lifts.

### Proof

For `a=(q-1)/2`, the q-arc is

```text
[1/2 - 1/q, 1/2].
```

For `a=(q+1)/2`, the q-arc is

```text
[1/2, 1/2 + 1/q].
```

By Lemma 3.1, the nearest old endpoint to `1/2`, other than `1/2` itself, is at
distance at least `1/p_k`. Since `q > p_k`,

```text
1/q < 1/p_k.
```

Thus each central q-arc reaches only into a one-sided interval on which the old
covered/uncovered state is constant until the next old endpoint. If the side is
covered, there is no residual component inside that q-arc. If the side is
uncovered, the residual component on that side extends beyond the q-arc. Any
residual component elsewhere is outside the q-arc. If there is no residual
component anywhere, then there is no birth. Therefore neither central remainder
can produce a closed or strict q-birth lift. ∎

## 5. Endpoint coincidence

### Lemma 5.1: old/new endpoint coincidence is only central

Let `p <= p_k` be an old prime and let `q` be any odd prime with `q>p_k`. If an
old p-arc endpoint is equal in `T` to a new q-arc endpoint, then the common
endpoint is `1/2`. In that case the q-remainder is one of the central remainders

```text
(q-1)/2, (q+1)/2.
```

### Proof

For odd `p`, write the old endpoint and new endpoint in canonical representatives
as

```text
m/(2p) = n/(2q),
```

where

```text
m in {1,3,...,2p-1},
n in {1,3,...,2q-1}.
```

Then

```text
mq = np.
```

Because `p` and `q` are distinct odd primes, `p|m` and `q|n`. The stated ranges
force

```text
m=p,     n=q.
```

Hence the common point is `1/2`.

For `p=2`, the old endpoints are `1/4` and `3/4`. An odd-q endpoint has the form
`n/(2q)` with `n` odd. Neither `n/(2q)=1/4` nor `n/(2q)=3/4` is possible for odd
prime `q`. Thus the only old/new endpoint coincidence is the central point
`1/2`.

If a q-arc has endpoint `1/2`, then its remainder is `(q-1)/2` or `(q+1)/2`.
∎

## 6. Closed terminal containment reduces to strict terminal containment

### Proposition 6.1: no closed-only endpoint-touch birth cases

Assume `R_k(r)` is nonempty. If `(r,a)` is closed q-terminal, then `(r,a)` is
strictly q-terminal.

Equivalently, in this later-prime PRC arc model, closed terminal containment
gives no additional nonempty q-birth lifts beyond strict terminal containment.

### Proof

Assume

```text
closure(R_k(r)) subset I_q(a)
```

but

```text
closure(R_k(r)) not subset int(I_q(a)).
```

Then `closure(R_k(r))` meets an endpoint of `I_q(a)`. Let that meeting point be
`x`.

The point `x` cannot lie in the interior of a residual component. If it did, a
small neighborhood of `x` would lie in `R_k(r)`, but since `x` is an endpoint of
`I_q(a)`, one side of that neighborhood would lie outside `I_q(a)`, contradicting
`closure(R_k(r)) subset I_q(a)`.

Therefore `x` is a boundary point of the residual set. Since

```text
R_k(r) = T \ U_k(r)
```

and `U_k(r)` is a finite union of closed old arcs, every boundary point of
`R_k(r)` is an old endpoint. Thus `x` is both an old endpoint and a new q-endpoint.

By Lemma 5.1, `x=1/2`, and `a` is one of the two central remainders. By Lemma
4.2, the central remainders do not produce closed or strict q-birth lifts. This
contradicts the assumption that `R_k(r)` is nonempty and closed q-terminal.

Hence closed q-terminal containment implies strict q-terminal containment for
nonempty residual sets. ∎

## 7. Abstract circle-arc obstruction lemma

### Lemma 7.1: short arcs cannot bridge separated residual components

Let `U` be a finite union of closed arcs on `T`, and let `R=T\U`. Assume every
nonempty connected component of `U` has circular length at least `delta`. Let
`J` be a closed circular arc of length `lambda < delta`. Then `J` cannot contain
the closures of two or more connected components of `R`.

### Proof

Suppose `J` contained the closures of at least two connected components of `R`.
Choose a cut outside `J` and use a local linear representative in which `J` is a
single closed interval. Among the residual components contained in this
representative, choose two that are adjacent in the induced linear order. No
residual component lies between them, so the closed interval between their
closures is a connected component of `U`.

The arc `J` contains that separating covered component. By hypothesis, the
covered component has length at least `delta`, while `J` has length
`lambda < delta`. This is impossible. ∎

## 8. PRC covered component lower bound

### Lemma 8.1: every old covered component is at least as long as the smallest old arc

Every nonempty connected component of `U_k(r)` has circular length at least

```text
1/p_k.
```

### Proof

The set `U_k(r)` is a finite union of old p-arcs. Every nonempty connected
component of this union contains at least one whole old p-arc. Each old p-arc has
length `1/p_i`, and since `p_i <= p_k`,

```text
1/p_i >= 1/p_k.
```

Therefore every nonempty connected component of `U_k(r)` has circular length at
least `1/p_k`. ∎

## 9. Multi-gap obstruction

### Lemma 9.1: one q-arc cannot close multiple residual components

If `R_k(r)` has at least two connected components, then no q-remainder `a` gives
a closed or strict q-birth lift.

### Proof

A new q-arc has circular length

```text
1/q.
```

Since `q>p_k`,

```text
1/q < 1/p_k.
```

By Lemma 8.1, every nonempty connected component of `U_k(r)` has circular length
at least `1/p_k`. Apply Lemma 7.1 with

```text
U = U_k(r),   R = R_k(r),   J = I_q(a),
delta = 1/p_k,   lambda = 1/q.
```

No q-arc can contain the closures of two or more residual components. Therefore
if `R_k(r)` has at least two connected components, no q-remainder can make the
row closed or strictly q-terminal. ∎

## 10. Single-gap aperture classification

Assume now that `R_k(r)` has exactly one connected component. Write this
component in a local linear representative as

```text
G = (L,R).
```

The notation `R` here denotes the right endpoint of the single gap; it is not the
set `R_k(r)`.

### Lemma 10.1: strict containment is equivalent to the aperture inequality

For a q-remainder `a`, choose a lifted representative

```text
tilde a in a + qZ
```

and a local linear representative in which both `G` and the relevant q-arc do not
cross the cut. Such a representative is the local form of choosing compatible
lifts of the gap and q-arc in the universal cover for the containment test. Then

```text
closure(G) subset int(I_q(a))
```

if and only if

```text
qR - 1/2 < tilde a < qL + 1/2.
```

### Proof

In the chosen representative,

```text
I_q(tilde a) = [ (tilde a - 1/2)/q, (tilde a + 1/2)/q ].
```

Strict containment of `closure(G)=[L,R]` in the interior of this q-arc is exactly

```text
(tilde a - 1/2)/q < L < R < (tilde a + 1/2)/q.
```

Multiplying by `q` gives

```text
tilde a - 1/2 < qL
qR < tilde a + 1/2.
```

Equivalently,

```text
qR - 1/2 < tilde a < qL + 1/2.
```

∎

Throughout Section 10, `L` and `R` are lifted real coordinates in the universal
cover, chosen so that `L<R`. If the residual gap crosses the `0`-cut in `T`, the
chosen lift may have `R>1`.

### Definition 10.2: strict aperture window

The open interval

```text
A_q(G) = (qR - 1/2, qL + 1/2)
```

is the strict aperture window of the single residual gap `G`.

### Lemma 10.3: uniqueness of the q-remainder candidate

For a nonempty single residual gap `G=(L,R)`, the strict aperture window contains
at most one integer. Therefore a single-gap row has at most one q-remainder that
can produce a q-birth lift.

### Proof

The width of the aperture window is

```text
(qL + 1/2) - (qR - 1/2)
= 1 - q(R-L)
= 1 - q length(G).
```

If this width is non-positive, the aperture window is empty and contains no
integer. If this width is positive, then since `G` is nonempty,
`length(G)>0`, and therefore

```text
1 - q length(G) < 1.
```

An open interval of width strictly less than `1` contains at most one integer.
That integer, if it exists, determines a unique q-remainder modulo `q`. ∎

## 11. Main theorem

### Theorem 11.1: General q-Prime Single-Gap Aperture Classification

Fix `k>=3`, an old residue `r in Z/M_kZ`, and any odd prime `q` with `q>p_k`.
Let `R_k(r)` be the old residual set.

1. If `R_k(r)` is empty, the old row is already terminally closed, so no q-birth
   lift is counted.

2. If `R_k(r)` has two or more connected components, no q-remainder `a` produces
   a closed or strict q-birth lift.

3. If `R_k(r)` has exactly one connected component `G=(L,R)`, then a q-remainder
   `a` produces a closed q-birth lift if and only if it produces a strict q-birth
   lift, and this occurs if and only if there exists a lifted representative

   ```text
   tilde a in a + qZ
   ```

   such that

   ```text
   qR - 1/2 < tilde a < qL + 1/2.
   ```

4. The aperture window

   ```text
   (qR - 1/2, qL + 1/2)
   ```

   contains at most one integer. Therefore a single-gap old row has at most one
   q-remainder that can produce a q-birth lift.

### Proof

If `R_k(r)` is empty, there is no old residual set to be newly closed, so no
birth lift is counted.

If `R_k(r)` has at least two connected components, Lemma 9.1 rules out closed and
strict q-birth lifts.

If `R_k(r)` has exactly one connected component `G`, then row-level terminal
containment is the same as terminal containment of that single component. By
Proposition 6.1, closed terminal containment is equivalent to strict terminal
containment for nonempty residual sets in this model. By Lemma 10.1, strict
terminal containment is equivalent to the aperture inequality

```text
qR - 1/2 < tilde a < qL + 1/2.
```

Finally, Lemma 10.3 proves that the aperture window contains at most one integer,
so the q-remainder candidate is unique if it exists. ∎

## 12. Next-prime corollary and consequences

### Corollary 12.1: next-prime PRC birth classification

Taking `q=p_{k+1}` in Theorem 11.1, every next-prime q-birth lift in the PRC
circular-arc model is a single-gap aperture-alignment event.

### Corollary 12.2: multi-gap birth is structurally impossible

In the stated PRC circular-arc model, a q-birth lift cannot arise from
simultaneously closing two or more old residual gaps. Any q-birth lift must be a
single-gap aperture event.

### Corollary 12.3: birth requires q-grid phase alignment

For a single-gap old row, the gap length condition

```text
length(G) < 1/q
```

is necessary but not sufficient. A q-birth lift occurs exactly when the strict
aperture window contains an integer representative of a q-remainder. Thus birth
requires q-grid phase alignment, not merely a short residual gap.

### Corollary 12.4: one candidate per single-gap row

A single-gap old row has at most one q-remainder candidate that can produce a
q-birth lift.

## 13. Relation to PRC v2.3/v2.5 terminology

This theorem uses the following terminology mapping.

```text
old p-arc      = old p-cloud, when earlier artifacts use cloud language
new q-arc      = new q-cloud
old residual component = old residual gap
q-birth lift   = later-prime lifted row with nonempty old residual set that becomes terminal
```

The theorem is stated for the mathematical terminal-containment definitions in
Section 2. It does not, by itself, claim that every implementation-level checker,
CSV field, or historical artifact label is identical to these definitions.
Implementation-level equivalence should be checked by a separate exact-audit
script.

The existing v2.7 exact audit checks the committed recorded birth rows in the
finite next-prime support CSV for `B4->B5`, `B5->B6`, and `B6->B7` for consistency
with this theorem's next-prime corollary. It is not a full finite-universe
completeness audit. That audit is implementation evidence; it is not the proof
of the general q-prime theorem.

## 14. Public Theorem Boundary

This theorem note claims a structural classification inside the stated PRC
circular-arc model:

```text
nonempty q-birth lift  <=>  single residual gap + q-grid aperture alignment.
```

This theorem is general over all old residues and all later odd prime moduli
`q>p_k` in the PRC circular-arc model. Implementation-level PRC artifacts are
covered only after exact audit confirms that their `Close(row)` predicate
matches the closed terminal-containment definition used here.

For later primes beyond the next-prime case, this is a direct q-lift theorem
over the old prefix. It is not a claim about the full sequential insertion of all
intermediate primes.

It does not claim:

```text
- a B8 theorem,
- a full transition-graph theorem,
- an asymptotic law,
- a prime-gap theorem outside the PRC model,
- a general predictor for future rows,
- or mechanical proof verification.
```

Finite exact audits are useful for validating implementations and examples, but
they are not the proof of the theorem. The proof of the theorem is the geometric
argument given above.
